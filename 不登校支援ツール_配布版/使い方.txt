# 不登校支援ツール 使い方

## 起動方法

「不登校支援ツール」アイコンをダブルクリック

## 使い方

1. **基本情報を入力**
   - 児童氏名
   - 保護者氏名
   - 学校名
   - 学年

2. **面談メモを入力**
   - 会話の流れのまま自由に記録してください
   - AIが自動で整理します

3. **AI分析を実行**
   - 「🔍 AI分析を実行」ボタンをクリック
   - 30秒〜1分ほどお待ちください

4. **結果を確認**
   - アセスメントシート：「💾 Excelに出力」ボタン
   - 報告書テキスト：「📋 全文をコピー」ボタン

## 保存場所

作成したExcelファイルは以下に保存されます：

```
マイドキュメント > 不登校支援ツール > output
```

## よくある質問

**Q: 起動しません**
A: ダブルクリックしてしばらく待ってください（初回は少し時間がかかります）

**Q: AI分析が終わらない**
A: インターネット接続を確認してください

**Q: 以前作ったファイルが見つからない**
A: マイドキュメント > 不登校支援ツール > output フォルダを確認してください
