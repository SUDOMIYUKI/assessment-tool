==========================================
  不登校支援ツール - 配布版
==========================================

【クイックスタート】

1. 「不登校支援ツール.exe」をダブルクリック
2. 児童情報を入力
3. 面談メモを記録
4. 「AI分析を実行」をクリック
5. 完成！

【特徴】

✅ インストール不要
✅ ダブルクリックですぐ起動
✅ 自動でアセスメントシート生成
✅ 報告書テキストも自動作成
✅ オフライン動作（AI分析時のみネット接続）

【保存場所】

作成したファイルは以下に保存されます：

  マイドキュメント → 不登校支援ツール → output

【システム要件】

- Windows 10/11
- インターネット接続（AI分析時）
- 空き容量: 100MB以上

【サポート】

詳しくは「使い方.txt」と
「サポート連絡先.txt」をご覧ください。

【初回起動時の注意】

初回は起動に10〜30秒かかる場合があります。
気長にお待ちください。
